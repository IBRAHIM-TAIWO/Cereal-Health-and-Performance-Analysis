# Cereal-Health-and-Performance-Analysis
This project analyzed a cereal dataset using Excel to evaluate product healthiness, customer ratings, and areas needing improvement.

Key insights include:
• Only a small portion of cereals meet healthy standards
• High sugar is the primary issue across most products
• Higher sugar cereals tend to receive higher customer ratings — highlighting a health vs taste trade-off

Tools used:
Excel

process covered:
Data cleaning
pivot tables
KPIs
dashboard design
and business storytelling
